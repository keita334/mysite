# FilamentMaker

`FilamentMaker` は、フィラメント作成に関するファイルを管理するディレクトリです。

## 目的

- フィラメント作成プロセスの記録
- 設定値や実験結果の整理
- 関連データや手順の再利用

## 想定構成

- `settings/`: 温度・速度・材料などの設定
- `logs/`: 試作ログ、結果メモ
- `assets/`: 参考画像、図、補助データ

## 使い方

1. 作業前に設定を記録する
2. 作業ごとに結果をログとして残す
3. 再現可能な条件をREADMEやログに追記する
