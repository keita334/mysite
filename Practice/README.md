# Practice

このフォルダは、Web系企業のコーディングテスト（Track想定）の練習用です。

## 使い方
- 問題文や要件をここに追加してください（例: `problem.md`）。
- 解答コードをこのフォルダ配下で管理してください。
- 必要に応じて、私に以下を依頼してください。
  - 問題の要件整理
  - 解法の方針比較（計算量・可読性）
  - 実装サポート
  - リファクタ・レビュー
  - テストケース作成

## 推奨ファイル構成（例）
- `problem.md` : 問題文・制約
- `solution.*` : 解答コード
- `test_cases.md` : テスト観点
- `notes.md` : 振り返り
